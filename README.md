# UNITY-VR

Advanced Unity Virtual Reality (VR) project focused on interactive visualizations of human biological systems.

## 📋 Overview

An educational and research-focused project for building virtual reality applications using the Unity engine, with emphasis on:
- Visualization of human health and biological systems
- Interactive simulations in VR environments
- Advanced graphics rendering techniques

## 🎯 Main Projects

The repository contains several sub-projects:

### 1. **Heart_VR** ❤️
Interactive simulation of the heart and cardiovascular system in VR
- 3D visualization of the heart
- Blood flow simulation
- Full user interaction and manipulation

### 2. **CUTTING** ✂️
Surgery and dissection simulation tool in VR
- Realistic cutting and surgical simulation
- Advanced physical interactions

### 3. **bloodf_low** 🩸
Blood flow and circulatory system simulation
- Dynamic visualization of blood movement
- Realistic physical models

### 4. **nasal cavity** 👃
Interactive simulation of nasal cavity and upper respiratory system
- Detailed visualization of nasal passages
- Accurate anatomical models

### 5. **squueze** (squeeze) 🤏
Compression and compressive movement simulation
- Realistic physical interactions
- Squeezing and movement simulation

## 🛠️ Technical Requirements

### Required Environment:
- **Unity Engine** (compatible version)
- **XR Hands** 1.6.1 or newer
- **C# 7.3+**

### Libraries and Packages:
- XR Hands Package (for hand interaction in VR)
- XR Interaction Toolkit
- Universal Render Pipeline (URP) / HDRP

## 🚀 Quick Start

### Installation:

1. **Clone the repository:**
```bash
git clone https://github.com/mohamedhamdy9707-byte/UNITY-VR.git
cd UNITY-VR
```

2. **Open the project in Unity:**
   - Open Unity Hub
   - Click "Open Project"
   - Select the `Heart_VR` folder or any sub-project

3. **Install required packages:**
   - Open `Window > TextMesh Pro > Import TMP Essentials`
   - Install required XR packages from Package Manager

### Running:

1. Select a start scene from the Scenes folder
2. Press the Play button in the Unity editor
3. Use your VR controller or mouse to interact
